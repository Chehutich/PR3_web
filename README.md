# Практична робота №3

## Тема: Основи Node.js та HTTP

Цей проект представляє собою базовий сервер на Node.js з використанням вбудованого модуля `http`.

### Реалізовані рівні:
- **Base**: Створено сервер на модулі `http`, який віддає статичний JSON-список подій за маршрутом `/api/events`.
- **Middle**: Реалізовано логування кожного запиту у консоль. Конфігурація (наприклад, порт) винесена у окремий файл `config.js`.
- **Advanced**: Налаштовано `nodemon` для автоматичного перезапуску сервера під час розробки (команда `npm run dev`). Додано обробку помилок (статус-коди 404 для невідомих маршрутів та 500 для помилок сервера).

### Інструкції з запуску

1. Встановіть залежності (необхідні для `nodemon`):
   npm install

2. Для запуску сервера в режимі розробки (з автоматичним перезапуском при зміні файлів):
   npm run dev

3. Для звичайного запуску сервера:
   npm start

### Структура проекту
- `index.js` — головний файл сервера.
- `config.js` — модуль з налаштуваннями конфігурації.
- `data.json` — статичні дані подій.
- `package.json` — конфігурація проекту та скрипти запуску.
